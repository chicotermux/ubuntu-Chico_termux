# ubuntu-Chico_termux

#Instalación

$ apt update

$ apt upgrade

$ git clone https://github.com/chicotermux/ubuntu-Chico_termux

$ cd ubuntu-Chico_termux

$ ./ubuntu_Chico-Termux

Haciendo eso se iniciará Ubuntu

# Abrir

$ cd Chico-termux_Ubuntu

$ ./start-ubuntu.sh

# salir

$ éxito
